# 摄影装备选购与参数详解

> 相机参数一堆数字看不懂？这篇从实用角度拆解关键参数，帮你看懂规格表、做出正确选购决策。

## 一、相机类型选择

### 主流类型对比

| 类型 | 优势 | 劣势 | 适合人群 |
|------|------|------|----------|
| **微单（无反）** | 轻便、新技术、实时取景 | 电池续航稍短 | 绝大多数人的首选 |
| **单反（DSLR）** | 光学取景、续航好、生态成熟 | 机身重、技术老化 | 有存量镜头或特殊需求 |
| **卡片机** | 极度便携 | 画质受限、可玩性低 | 随身记录 |
| **手机** | 永远在身边、计算摄影强 | 暗光和大背景虚化受限 | 日常记录 |

### 画幅对比（重要！）

```
全画幅（36×24mm）
  ├── 感光面积最大 → 暗光最好
  ├── 背景虚化最强
  ├── 镜头选择最丰富
  └── 价格最贵（机身 1-3万+）

APS-C（约 23×15mm）
  ├── 性价比最高
  ├── 等效焦距 ×1.5（尼康/索尼）或 ×1.6（佳能）
  ├── 入门到中端的最佳选择
  └── 价格（3千-1万5）

M4/3（17×13mm）
  ├── 体积最小巧（奥林巴斯/松下）
  ├── 等效焦距 ×2
  └── 视频、便携性好

中画幅（> 全画幅）
  └── 商业摄影专用，价格极贵
```

## 二、核心参数详解

### 2.1 传感器像素

**误区**：像素越高画质越好 ❌

**实际情况**：
- 2400万像素已足够 A4 打印和大屏显示
- 像素高 ≠ 暗光好（暗光由单位像素面积决定）
- 高像素文件更大，对电脑要求更高

**建议**：
- 日常拍摄：2000-3000万像素足够
- 风光/商业需要大量裁切：5000万+

### 2.2 ISO（感光度）

```
低 ISO（100-400）：  充足光线，画质最纯净
中 ISO（800-3200）： 室内/阴天，可接受
高 ISO（6400+）：    暗光场景，噪点明显增加

关键指标：可用 ISO 上限
  入门机： ISO 3200 可用
  中端机： ISO 6400-12800 可用
  全画幅旗舰： ISO 25600+ 可用
```

### 2.3 快门速度

```
快门速度影响：
  运动模糊 vs 凝固动作
  手持稳定性（安全快门 = 1/焦距）

常用参考值：
  1/1000s 以上  凝固快速运动（鸟类、运动员）
  1/250s        日常人像
  1/60s         静态场景手持极限（广角）
  1/30s 以下    需要三脚架
  1-30s         夜景、光绘、星轨
  B门（bulb）   长曝光（天文摄影）
```

### 2.4 光圈（F 值）

```
光圈 f/1.4  ← 大光圈（进光多，景深浅，虚化强）
光圈 f/2.8
光圈 f/4
光圈 f/5.6
光圈 f/8    ← 最佳画质区间
光圈 f/11
光圈 f/16   ← 小光圈（进光少，景深深，适合风景）
光圈 f/22   ↓ 开始出现衍射，画质下降

记住：F 数字越小 = 光圈越大！
```

### 2.5 对焦系统

| 技术 | 说明 | 优势 |
|------|------|------|
| 相差对焦（PDAF）| 相素专门用于检测相差 | 快速准确 |
| 反差对焦 | 寻找对比度最大值 | 精确但较慢 |
| 混合对焦 | PDAF + 反差 | 兼顾速度与精度 |
| AI 识别对焦 | 识别人眼、动物、车辆等 | 现代相机标配 |

**眼部识别（Eye AF）**：人像拍摄的革命性功能，微单普遍支持，强烈推荐优先考虑。

## 三、镜头选购

### 3.1 焦距与视角

```
超广角 < 24mm   风景、建筑、星空、VLOG
广角   24-35mm  日常扫街、环境人像
标准   50mm     最接近人眼视角，万能
中焦   85mm     人像黄金焦距，压缩感好
中长   135mm    人像、体育
长焦   200-600mm 野生动物、运动赛事、月亮
```

### 3.2 入门推荐搭配（APS-C 机）

```
第一支镜头：
  套头（18-55mm）→ 免费/低价获取，了解焦距感觉
  
第二支镜头（升级）：
  35mm f/1.8  → 日常、人文、人像（等效 52mm，接近标准）
  50mm f/1.8  → 人像虚化入门（等效 75mm，适合小空间）
  
拓展：
  广角：10-20mm  → 风景、建筑
  长焦：70-300mm → 野生动物、旅行
```

### 3.3 镜头参数看什么

```
最大光圈   f/1.8、f/2.8 等 → 越小越好（但越贵越重）
最近对焦距离 → 越短越适合近拍
防抖（IS/OIS）→ 手持慢速快门必要
镀膜      → 影响逆光耀斑和色彩还原
```

## 四、配件优先级

```
必备（高优先级）：
  ✅ 存储卡   — 选 UHS-I V30 以上，读速 100MB/s 以上
  ✅ 备用电池  — 至少 2 块
  ✅ UV 保护滤镜 — 保护镜头前片

建议（中优先级）：
  ✅ 三脚架   — 夜景、长曝光必备，不要买太便宜的
  ✅ 快门线   — 避免触碰相机抖动
  ✅ 镜头清洁工具 — 气吹 + 清洁液 + 镜头纸

进阶（低优先级）：
  ✅ ND 滤镜  — 大光圈拍水流瀑布
  ✅ 偏振镜（CPL）— 压蓝天、消反光
  ✅ 闪光灯  — 室内外补光
```

## 六、专业术语速查

| 术语 | 解释 |
|------|------|
| **画幅** | 感光元件（传感器）的尺寸，全画幅（36×24mm）> APS-C > M43，越大暗光越强 |
| **光圈（Aperture）** | 镜头进光口大小，用 f 值表示，f/1.8 比 f/8 大，大光圈进光多、背景虚化强 |
| **快门速度** | 感光元件曝光时间，1/1000s 可以定格运动，1/30s 拍夜景但需三脚架 |
| **ISO** | 感光度，越高越亮但越噪点，白天用 100-400，夜景可能需要 3200+ |
| **焦距（Focal Length）** | 镜头的视角参数，数字越小视角越广，越大越远，50mm 接近人眼视角 |
| **等效焦距** | APS-C 相机焦距需要乘以 1.5（或 1.6）才能换算成全画幅的视角效果 |
| **景深** | 照片中清晰对焦的范围，大光圈/长焦距/近距离拍摄会产生浅景深（虚化背景） |
| **RAW 格式** | 相机传感器原始数据，未经处理，保留最多后期空间，文件比 JPEG 大 5-10 倍 |
| **防抖（IS/OSS/VR）** | 补偿手持抖动的技术，分镜头防抖和机身防抖（IBIS），手持拍摄很有用 |
| **自动对焦（AF）** | 相机自动识别对焦点，现代相机的人脸/人眼 AI 追焦非常好用 |
| **白平衡（WB）** | 校正不同光源下的色偏，让白色物体看起来是白色而非橙/蓝色 |
| **直方图** | 显示照片明暗分布的图表，偏左曝光不足，偏右曝光过度，参考比液晶屏更准 |
| **色彩空间（Color Space）** | sRGB 适合网络发布，Adobe RGB 色域更广适合印刷，日常用 sRGB 够了 |
| **ND 滤镜** | 减光镜，让大光圈在强光下也能慢速快门，拍丝滑流水/轨迹光常用 |
