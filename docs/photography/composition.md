# 构图原则

好的构图能让照片更具视觉冲击力和艺术感。以下是几种最实用的构图技巧。

将画面用两条横线和两条竖线分成九宫格，**主体放在交叉点或线上**。

- 人物眼睛放在上 1/3 线
- 地平线放在上或下 1/3 线，避免居中
- 打开相机的网格线辅助练习

## 2. 引导线 (Leading Lines)

利用画面中的线条引导观众视线到主体。

**常见引导线**：
- 道路、铁轨、河流
- 栏杆、走廊、桥梁
- 树枝、光线、阴影

## 3. 框架构图 (Framing)

用前景元素框住主体，增加层次感和聚焦效果。

**自然框架**：
- 门洞、窗户
- 树枝形成的空隙
- 建筑结构

## 4. 对称与平衡

- **对称构图**：建筑倒影、镜面反射，给人稳定庄严感
- **不对称平衡**：大小主体分布在画面两侧，视觉重量相当

## 5. 留白 (Negative Space)

给主体周围留出空白，营造简约、孤独、宁静的氛围。

适合：
- 极简风光
- 人物情绪表达
- 产品摄影

## 6. 前景运用

在画面前景加入元素，增加**纵深感**和**立体感**。

技巧：
- 低角度靠近前景物体
- 使用广角镜头夸张透视
- 大光圈虚化前景
