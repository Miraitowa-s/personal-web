# Lightroom 后期调色完全指南

> 掌握这套工作流，让每张照片都有自己的情绪。

---

## 一、为什么要后期？

很多人觉得"好照片不需要后期"，但这是个误解。相机记录的 RAW 文件本质上是一堆未处理的传感器数据，后期是**把信息变成作品**的过程——就像暗房时代的冲洗和印放一样，它一直都是摄影的一部分。

---

## 二、Lightroom 的核心工作流

### 1. 基础调整（Basic Panel）

按这个顺序处理，逻辑最清晰：

| 参数 | 作用 | 常见操作 |
|---|---|---|
| **曝光 Exposure** | 整体亮度 | 如果直方图堆在两端，先修它 |
| **高光 Highlights** | 压暗过曝区域 | 人像亮部拉到 -40 ~ -70 |
| **阴影 Shadows** | 提亮暗部细节 | +20 ~ +50 找回暗部层次 |
| **白色 Whites** | 最亮处上限 | 微调，防止死白 |
| **黑色 Blacks** | 最暗处下限 | 微向左拉，增加对比感 |
| **清晰度 Clarity** | 中间调对比 | 人像慎用，风光可 +20 |
| **自然饱和度 Vibrance** | 智能饱和，保护肤色 | 比 Saturation 好用 |

### 2. 色调曲线（Tone Curve）

曲线比滑块精细，但更难控制。记住这几个经典形状：

- **S 形曲线**：提高整体对比度，最常用
- **提亮阴影**：把曲线左下角向上拖，让暗部不死黑（胶片感的来源）
- **压暗高光**：右上角向下压，保留天空细节

```
提示：按住 Cmd/Ctrl 点击画面中某个区域，
     曲线上会出现对应亮度的控制点，更直观。
```

### 3. HSL 色板——精准控制某种颜色

HSL = **Hue（色相）/ Saturation（饱和度）/ Luminance（明度）**

常见用法：
- 把草地绿调向黄绿，显得更有活力
- 把天空蓝拉深，不用偏光镜也能有感
- 压低橙色明度 → 人像肤色更立体

### 4. 色彩分级（Color Grading）

原来叫"分离色调"，现在功能更强。

- **高光**加暖色（橙黄）→ 阳光感
- **阴影**加冷色（蓝青）→ 增加层次感
- 两者反向就是经典的"橙青色调"（电影感来源）

## 三、常用色调风格

### 🎞️ 胶片感
1. 阴影提亮（Blacks +15 ~ +30）
2. 曲线左下角上移
3. 高光加暖色，阴影加冷色
4. 降低对比度，降低饱和度
5. 加颗粒（Effects → Grain）

### 🌅 日系清透
1. 曝光 +0.3 ~ +0.5
2. 高光 -20，阴影 +30
3. 色温偏暖 +200
4. 绿色 / 黄色饱和度降低
5. 整体饱和度 -10

### 🎬 电影青橙
1. 色彩分级：高光橙，阴影蓝青
2. 色调曲线 S 形加对比
3. 橙色饱和 +10，蓝色饱和 +15
4. 清晰度 +10

## 五、一些被忽视的好功能

- **蒙版（Masking）**：AI 自动选人、选天空、选主体，精准局部调整
- **修复画笔**：去掉传感器灰尘、皮肤瑕疵，比 PS 方便
- **虚拟副本**：同一张照片试不同色调，不占额外空间
- **预设（Presets）**：做好自己的预设，下次一键应用，保持风格统一

---

## 六、工作流总结

```
导入 → 筛片（P/X 标记）→ 基础调整 → 色调曲线 
→ HSL → 色彩分级 → 局部调整（蒙版）→ 锐化/降噪 
→ 镜头校正 → 导出
```

> **最重要的原则**：后期是为了呈现你当时感受到的画面，而不是弥补拍摄的失误。
