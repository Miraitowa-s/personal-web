## 🛠️ Lightroom 核心工作流

### 1. 导入与整理
```markdown
**文件夹结构推荐：**
- 📁 RAW 素材库
  - 📁 年份
    - 📁 2026-04-09_樱花季
      - 📄 DSC_001.NEF
      - 📄 DSC_002.NEF
    - 📁 2026-04-10_城市夜景
      - 📄 DSC_101.NEF

**关键字标签系统：**
- 地点：`#北京` `#故宫` `#颐和园`
- 主题：`#人像` `#风光` `#建筑`
- 季节：`#春季` `#樱花` `#黄昏`
- 技术：`#长曝光` `#HDR` `#黑白`
```

### 2. 基础调整流程
```markdown
**建议调整顺序：**
1. **镜头校正**：启用配置文件校正、删除色差
2. **白平衡**：使用吸管工具点中性灰区域
3. **曝光**：直方图右侧不溢出，保留亮部细节
4. **对比度**：增加画面立体感
5. **高光/阴影**：找回细节，平衡明暗
6. **白色/黑色**：精细控制最亮/最暗部分
7. **清晰度**：增强中间调对比，但不过度
8. **鲜艳度**：智能增加欠饱和颜色
9. **饱和度**：谨慎使用，防止颜色过艳

**示例参数（人像）：**
- 曝光：+0.3
- 对比度：+10
- 高光：-30
- 阴影：+20
- 白色：+5
- 黑色：-5
- 清晰度：+5
- 鲜艳度：+10
```

### 3. 色彩分级实战
```markdown
**HSL/颜色调整：**
- **皮肤色调**：橙色色相偏黄(-5)，降低饱和度(-10)，提高明度(+15)
- **天空**：蓝色降低亮度(-15)，增加饱和度(+20)
- **植物**：绿色偏青(+10)，黄色偏绿(-5)

**分离色调（电影感）：**
- 高光：色相 45，饱和度 10
- 阴影：色相 220，饱和度 5
- 平衡：+25

**校准工具（胶片模拟）：**
- 红原色：色相 +15，饱和度 +10
- 绿原色：色相 +10，饱和度 +5
- 蓝原色：色相 -5，饱和度 +15
```

### 4. 局部调整技巧
```markdown
**渐变滤镜（风光）：**
1. 天空区域：曝光 -1.0，对比度 +15，高光 -30
2. 地面区域：曝光 +0.3，清晰度 +10

**径向滤镜（人像）：**
1. 面部区域：曝光 +0.5，对比度 +5，锐度 +20
2. 背景虚化：清晰度 -30，饱和度 -15

**画笔工具（精细调整）：**
- 眼神光：新建画笔，大小 3，羽化 100，曝光 +0.8
- 皮肤柔化：清晰度 -20，锐度 -10
- 局部提亮：曝光 +0.3-0.7，范围控制
```

## 🎭 Photoshop 高级精修

### 1. 人像精修流程
```markdown
**磨皮处理（保留质感）：**
1. **高低频磨皮**：
   - 低频层：高斯模糊 8px
   - 高频层：应用图像（低频）- 线性光
   - 低频层：修复画笔修复瑕疵
   - 合并后，不透明度 50-70%

2. **中性灰磨皮**（商业级）：
   - 新建图层，填充 50% 灰色，混合模式：柔光
   - 画笔工具（前景色黑白，不透明度 5%）
   - 提亮暗部，压暗高光，统一肤色

3. **DR3/DR4插件**（效率提升）：
   - 自动识别皮肤区域
   - 智能磨皮，保留细节
   - 一键美白、润色
```

### 2. 风光合成技巧
```markdown
**曝光混合（HDR 效果）：**
1. **手动混合**：
   - 不同曝光 RAW 导入 LR，同步设置
   - 导出为 TIFF，在 PS 中堆栈
   - 图层蒙版，画笔擦出不同区域

2. **曝光混合插件**：
   - Aurora HDR：一键生成自然 HDR
   - Luminar AI：AI 自动平衡曝光
   - Nik Collection：色调映射工具

**景深合成（微距/风光）：**
1. 对焦点不同的多张照片
2. PS：文件 → 脚本 → 将文件载入堆栈
3. 全选图层，编辑 → 自动对齐图层
4. 编辑 → 自动混合图层（堆叠图像）
```

### 3. 色彩管理实战
```markdown
**工作流程：**
1. **拍摄阶段**：使用 RAW 格式，保留最大信息
2. **调整阶段**：LR 中使用 ProPhoto RGB 色彩空间
3. **输出阶段**：根据用途转换色彩空间
   - 网络：sRGB
   - 印刷：Adobe RGB
   - 专业打印：ProPhoto RGB

**颜色检查：**
- 直方图：RGB 三通道分布均匀
- 溢色警告：View → Gamut Warning
- 软打样：View → Proof Setup → Custom
```

## 🎯 实战案例解析

### 案例1：人像精修（婚礼摄影）
```markdown
**原片问题：**
- 肤色偏黄，不够通透
- 面部有轻微瑕疵
- 眼神光不足
- 背景杂乱

**处理步骤：**
1. **LR 基础调整**：
   - 色温 4800，色调 +8
   - 曝光 +0.4，对比度 +12
   - 高光 -35，阴影 +20
   - 橙色饱和度 -15，明度 +20

2. **PS 精修**：
   - 高低频磨皮（高斯模糊 6px）
   - 新建曲线图层提亮眼神
   - 使用内容感知填充简化背景
   - 可选颜色调整红色，增加唇色

3. **最终效果**：
   - 肤色白皙通透
   - 皮肤有质感无塑料感
   - 眼神明亮有神
   - 背景简洁突出主体
```

### 案例2：风光合成（日出金山）
```markdown
**拍摄素材：**
- 曝光包围：-2EV, 0EV, +2EV 各一张
- 对焦堆栈：前景、中景、远景各一张

**处理流程：**
1. **LR 同步处理**：
   - 统一白平衡（色温 6500）
   - 应用镜头校正
   - 同步基础调整

2. **PS 合成**：
   - 曝光混合：手动蒙版融合
   - 景深合成：自动混合图层
   - 调色：Nik Collection 色调对比度

3. **局部增强**：
   - 阳光区域：径向滤镜 + 暖色调
   - 水面：降低清晰度，增加蓝色饱和度
   - 山体：提高对比度，增强纹理

**技术要点：**
- 日出时刻光比较大，需要精细的曝光混合
- 前景细节需要景深合成保证全清晰
- 色彩过渡要自然，避免 HDR 痕迹
```

## 🚀 效率提升技巧

### 批量处理工作流
```markdown
**LR 批量处理流程：**
1. **导入筛选**：使用星级（★）标记优秀照片
2. **同步设置**：调整一张，同步到类似场景
3. **预设应用**：创建针对不同场景的预设
4. **批量导出**：设置统一命名规则和质量参数

**PS 动作录制：**
1. 打开动作面板（Window → Actions）
2. 新建动作组和动作
3. 开始录制，执行操作序列
4. 停止录制，批量应用到其他图片
```

```markdown

- 分离色调：高光橙黄，阴影青蓝
- 暗角：数量 -30，中点 50

**小清新（日系风格）：**
- 曝光：+0.5~1.0
- 对比度：-10~-20
- 高光：-30~-50
- 阴影：+20~+40
- 色调曲线：提亮暗部
```

## 💡 最佳实践总结

### 十大黄金法则
1. **先整体后局部**：先调曝光色彩，再处理细节
2. **保留 RAW 原片**：永远保留原始文件
3. **适度原则**：避免过度处理，保持自然
4. **风格统一**：同一组作品保持一致性
5. **定期备份**：重要作品多重备份
6. **色彩管理**：了解色彩空间和校色
7. **效率优先**：建立标准化工作流
8. **不断学习**：关注新技术和风格
9. **设备维护**：定期校准显示器
10. **分享交流**：参与社区，互相学习
