# 曝光三角

曝光三角是摄影最基础的概念，由**光圈、快门速度、ISO** 三个要素组成，它们共同决定照片的曝光量。

## 1. 光圈 (Aperture)

用 **f 值**表示，如 f/1.8、f/2.8、f/8、f/16

| f 值 | 效果 |
|------|------|
| f/1.4 - f/2.8 | 大光圈，景深浅，背景虚化强，适合人像 |
| f/5.6 - f/8 | 中等光圈，画质最佳，适合日常拍摄 |
| f/11 - f/16 | 小光圈，景深深，适合风光摄影 |

> 💡 **记忆口诀**：f 值越小，光圈越大，进光越多，虚化越强

## 2. 快门速度 (Shutter Speed)

控制感光元件曝光的时间长短。

- **高速快门** (1/500s 以上)：冻结运动，适合体育、飞鸟
- **安全快门**：通常不低于 1/焦距，如 50mm 镜头至少 1/50s
- **慢速快门**：需要三脚架，适合流水、车轨、星空

## 3. ISO 感光度

ISO 越高，感光元件对光线越敏感。

| ISO 范围 | 适用场景 | 画质表现 |
|---------|---------|---------|
| ISO 100-400 | 光线充足 | 最纯净，噪点最少 |
| ISO 800-1600 | 阴天/室内 | 轻微噪点，可接受 |
| ISO 3200+ | 夜景/暗光 | 明显噪点，谨慎使用 |

## 三者关系

```
曝光量 = 光圈 × 快门速度 × ISO
```

改变其中一个，需要相应调整其他两个来保持曝光平衡。

**示例**：
- 光圈从 f/2.8 缩到 f/5.6（进光减半）→ 快门从 1/250s 降到 1/125s（曝光时间加倍）

## 练习建议

1. 用 **A档（光圈优先）** 拍同一主体，分别用 f/2.8 和 f/11，观察景深变化
2. 用 **S档（快门优先）** 拍流水，分别用 1/1000s 和 1s，观察动态效果
3. 在暗光环境下，尝试 ISO 400、1600、6400，对比噪点差异
