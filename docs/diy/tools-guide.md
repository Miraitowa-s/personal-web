# DIY 工具入门指南

> 从零开始搭建工具库：哪些工具必备，哪些等需要时再买，怎么安全使用。

## 一、工具选购原则

### 1.1 买好不买贵，买够不买多

```
误区：先买齐所有工具再开始做
正确：项目驱动，需要什么买什么

误区：便宜工具先凑合
正确：核心工具宁可少买几样，买质量好的
      好工具用 10 年，便宜货用 1 年坏掉
```

### 1.2 品牌参考

| 类别 | 预算品牌 | 品质品牌 | 专业品牌 |
|------|----------|----------|----------|
| 电动工具 | 大有、东成 | 博世（Bosch）、牧田 | 得伟（Dewalt）、米沃奇 |
| 手工具 | 史丹利 | 斯坦利（Stanley）Pro | 日本品牌（TAJIMAl等）|
| 螺丝刀 | 力易得 | 克莱森 | 维修宝、威哈 |

## 二、必备工具清单

### 第一批（基础工具，约 500-1000 元）

```
测量工具：
  ✅ 卷尺（5米）         — 任何项目都要量尺寸
  ✅ 直角尺              — 画直线、检查垂直
  ✅ 水平尺（磁吸）       — 挂东西必用

固定与连接：
  ✅ 榔头（中号）         — 钉钉子、组装敲击
  ✅ 螺丝刀套装           — 十字/一字，各种尺寸
  ✅ 钳子套装             — 尖嘴钳、平口钳、斜口钳

切割：
  ✅ 美工刀（大号）        — 开箱、划线、切软材料
  ✅ 手锯                 — 切木材

辅助：
  ✅ 铅笔 + 划线器        — 标记切割线
  ✅ 夹钳（F夹/G夹）2个   — 固定工件，粘合时保持位置
```

### 第二批（升级工具，约 500-1500 元）

```
电动工具（影响效率最大）：
  ✅ 电钻 + 螺丝刀二合一  — 打孔、拧螺丝，电动必备
     推荐：博世 GSR 12V   — 轻巧够用
  ✅ 电动砂纸机（随机轨道型）— 打磨省力，效果好

精度工具：
  ✅ 组合角尺              — 多功能划线
  ✅ 划线规                — 平行线划线
```

### 第三批（进阶工具，按需购买）

```
  电圆锯 / 曲线锯    — 大量直线/曲线切割
  木工凿 + 锤       — 榫卯、精细木工
  修边机（路由器）   — 边缘处理，铣槽
  空压机 + 气动钉枪  — 装修、大批量组装
  焊接工具          — 金属制作
```

## 三、工作空间布置

### 最小工作空间（阳台/储藏间）

```
必备：
  工作台（1.2m 以上长度）
  良好照明（LED 工作灯）
  插座排（至少 4 口）
  
收纳：
  工具挂板（pegboard）— 常用工具可见可取
  抽屉柜              — 小零件分类收纳
  
防护：
  地面铺橡胶垫或废旧地毯（减噪、防滑）
```

### 安全必备

```
个人防护装备（PPE）：
  ✅ 护目镜    — 切割、打磨时碎屑会飞溅
  ✅ 口罩      — 木屑、油漆粉尘防护（N95 最好）
  ✅ 手套      — 粗糙作业，但注意电动工具时不穿
     ⚠️ 使用旋转电动工具时不要戴手套（可能被卷入）
  ✅ 耳塞      — 角磨机、电锯噪音超过 85dB
```

## 四、常用材料速查

### 木材

| 类型 | 特点 | 适合项目 |
|------|------|----------|
| 实木板（松木）| 天然纹理，易加工 | 家具、装饰 |
| 多层板（胶合板）| 稳定不易变形 | 柜体、结构 |
| 密度板（MDF）| 表面平整，易上漆 | 柜门、展示架 |
| 木方（龙骨）| 便宜，做结构框架 | 隔断、背景墙 |

### 五金连接件

```
木螺丝：长度 = 穿过板厚 × 2.5
        粗细：3.5mm（薄板）/ 4mm（标准）/ 5mm（厚板）

膨胀螺丝：墙面固定，不同墙体选不同款
  混凝土墙 → 塑料膨胀管 + 自攻螺丝
  空心砖墙 → 蝴蝶型膨胀螺丝
  石膏板墙 → 石膏板专用螺钉（自带切入）

合页（铰链）：
  普通合页：工具箱、柜门
  液压铰链：橱柜门（阻尼缓冲）
  欧式铰链（暗合页）：现代家具柜门
```

### 粘合剂

| 粘合剂 | 适用 | 特点 |
|--------|------|------|
| 木工胶（白乳胶）| 木材 | 干后无色，强度高 |
| 万能胶 | 多材料 | 快干，柔性 |
| 环氧树脂 | 金属、石材 | 超强，防水 |
| 热熔胶 | 快速固定 | 临时固定，不耐热 |
| 结构胶 | 石材、金属 | 建筑级强度 |

## 五、基础工艺技能

### 打螺丝正确步骤

```
1. 钻引导孔（比螺丝略细）→ 防止木材开裂
2. 需要沉头时用沉头钻扩孔
3. 上木工胶（接合面）
4. 对准位置，手动旋入几圈定位
5. 电钻拧紧（不要过紧，木材会裂）
```

### 砂纸打磨顺序

```
粗磨（去掉凸起、刮痕）：80号
中磨（找平）：          120号
细磨（平滑）：          180-240号
精磨（上漆前）：        320号

规律：
  每次更换砂纸，必须去掉上一道的划痕才算完成
  顺木纹打磨，不要横向打磨
```

### 上漆/涂饰步骤

```
木材上漆（水性漆）：
  1. 打磨至 240 号
  2. 刷第一层底漆（或封闭剂）
  3. 干燥 4-6 小时
  4. 轻砂（400 号）去掉毛刺
  5. 刷第二层（漆）
  6. 重复步骤 4-5（共 2-3 层面漆）

上蜡（木蜡油）：
  1. 240 号砂纸打磨
  2. 用抹布涂薄薄一层蜡油
  3. 静置 30 分钟后抛光
  4. 24 小时后可使用
```

## 六、小项目入门推荐

### 项目 1：收纳木架（难度⭐）

```
材料：
  - 15mm 多层板，裁成需要尺寸
  - 螺丝、木工胶
工具：
  - 卷尺、直角尺
  - 手锯或电圆锯
  - 电钻
时间：半天
适合：完全新手的第一个项目
```

### 项目 2：储物凳（难度⭐⭐）

```
材料：
  - 18mm 多层板
  - 四条木腿或金属支架
  - 铰链、螺丝、木工胶
  - 可选：坐垫（泡沫+布料）
工具：
  - 电锯（切板）
  - 电钻
  - 夹钳
时间：1-2天
学习：切割精度、组合结构
```

### 项目 3：小书架（难度⭐⭐⭐）

```
材料：
  - 实木板（松木）
  - 圆榫 + 木工胶
  - 水性漆或木蜡油
工具：
  - 圆榫定位器
  - 夹钳（至少 4 个）
  - 砂纸、涂刷工具
时间：2-3天（含干燥时间）
学习：精准划线、圆榫接合、打磨上漆
```

## 七、专业术语速查

| 术语 | 解释 |
|------|------|
| **公差（Tolerance）** | 允许的尺寸误差范围，如 ±1mm，木工公差比金属加工宽松，但精度越高成品越好看 |
| **顺木纹/逆木纹** | 切割或打磨时与木纹方向的关系，顺纹更光滑，逆纹容易撕裂 |
| **开槽/开榫** | 在木材上切割特定形状用于连接，榫卯结构是传统木工连接方式，不用钉子 |
| **圆榫（Dowel）** | 圆柱形木棒，插入两块木材各自的孔中起定位和增强连接的作用 |
| **砂目（Grit）** | 砂纸的粗细规格，数字越小越粗（80#用于粗磨），数字越大越细（320#用于精磨） |
| **水性漆/油性漆** | 水性漆环保易干，适合室内；油性漆耐候性强，适合户外或潮湿环境 |
| **木蜡油** | 渗入木材的涂饰产品，保留木材质感，耐水耐污，适合实木家具 |
| **夹具/夹钳（Clamp）** | 固定工件的工具，木工胶固化期间必须施压，"永远嫌夹具不够用" |
| **划线（Marking）** | 在材料上精确标记切割线，是一切加工的起点，"划好线，锯好线" |
| **试装（Dry Fit）** | 上胶之前先不涂胶把所有部件组装测试，确认没有问题再正式粘合 |
| **锯路（Kerf）** | 锯片切削时损耗的材料宽度，手锯约 2-3mm，圆锯约 3mm，计算尺寸时需考虑 |
| **平整度** | 表面是否在同一平面，用金属直尺检验，新手最容易犯的错是接合处高低不平 |
