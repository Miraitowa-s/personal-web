# 知识库使用指南 📚

欢迎来到个人知识库！这里汇集了技术、创意和生活等多个领域的知识，采用多媒体形式呈现，包含文章、图片、视频等多种内容。

## 🎯 快速开始

### 1. 启动知识库
```bash
# 方法一：双击 start.bat（Windows）
# 方法二：手动启动
node server.js
```

### 2. 访问知识库
打开浏览器，访问：`http://localhost:3000/knowledge.html`

### 3. 主要功能
- **分类浏览**：8个专业分类，54篇文章
- **全文搜索**：实时搜索文章内容
- **大纲导航**：文章内部快速跳转
- **拖拽排序**：自定义分类和文章顺序
- **响应式设计**：支持电脑、平板、手机

## 🧭 使用导航指南

### 如何访问知识库首页？
在知识库页面，有**多种导航方式**：

1. **知识库Logo首页**（新功能）：
   - 点击左上角的 **"知识库"** Logo
   - 显示知识库概览页面，包含分类统计和使用说明

2. **返回网站首页**：
   - 点击右上角的 **"← 主页"** 按钮
   - 返回个人网站首页（`index.html`）

3. **浏览器导航**：
   - 后退按钮（←）或快捷键 **Alt+←** (Win) / **Cmd+←** (Mac)
   - 直接访问：`http://localhost:3000/knowledge.html`

### 知识库首页功能
点击知识库Logo后显示的概览页面包含：

- **📊 数据统计**：总文章数、分类数、知识领域
- **📁 分类结构**：技术类（38篇）vs 生活创意类（16篇）
- **🎯 使用说明**：浏览、搜索、返回操作指南
- **🗂️ 分类卡片**：点击分类卡片快速进入第一个文章

### 知识库内部导航
- **左侧分类栏**：点击分类名称展开/折叠文章列表
- **文章搜索**：顶部搜索框实时筛选内容
- **文章大纲**：右侧显示当前文章的 H2/H3 标题大纲
- **返回顶部**：右下角有 ↑ 按钮，点击可快速回到页面顶部
- **移动端侧边栏**：点击左上角 ≡ 图标打开/关闭侧边栏

### 内容组织结构
每个分类内部按照**学习路径优化排序**：
1. **指南类文章**（带🌟图标）：学习路径/基础概念指南
2. **核心教程**：主要技术/技能的核心内容

4. **实战演练**：实际项目/操作指南

## 📝 内容管理

### 添加新文章
1. 在 `docs/{分类}/` 目录下创建 `.md` 文件
2. 文件格式：使用标准 Markdown 语法
3. 支持多媒体嵌入：
   ```markdown
   ![图片描述](/image/example.jpg)
   
   <video width="80%" controls>
     <source src="/videos/example.mp4" type="video/mp4">
   </video>
   ```

### 更新目录
1. 修改 `knowledge.html` 中的 `FALLBACK_TREE`
2. 或依赖 `server.js` 自动扫描（优先）

### 添加多媒体资源
1. **图片**：放入 `/image/` 目录
2. **视频**：放入 `/videos/{分类}/` 目录
3. **音频**：放入 `/audio/` 目录（如有需要）

## 🛠 故障排除

### 常见问题
1. **服务器无法启动**
   - 检查 Node.js 是否安装：`node --version`
   - 检查端口是否被占用：`netstat -ano | findstr :3000`
   - 尝试修改端口：编辑 `server.js` 中的端口号

2. **页面加载缓慢**
   - 清除浏览器缓存
   - 检查网络连接
   - 减少同时打开的文章数量

3. **搜索功能失效**
   - 刷新页面重新加载
   - 检查控制台错误信息
   - 确保 `server.js` 正常运行

4. **图片/视频无法显示**
   - 检查文件路径是否正确
   - 确认文件格式支持（jpg/png/mp4）
   - 检查文件权限
