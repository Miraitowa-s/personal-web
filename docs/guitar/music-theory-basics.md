# 吉他乐理入门：不枯燥的基础知识

> 乐理不是数学，是语言。学它是为了更自由地说话。

---

## 一、为什么学乐理？

很多吉他手抗拒乐理，觉得"会弹就够了"。但乐理能让你：
- 看到一个和弦进行，就能推算出下一个应该是什么
- 听到一段旋律，就能找到对应的音在哪里
- 和其他乐手沟通时，有共同语言

**不需要学很多，学刚好够用的就行。**

---

## 二、十二个音

西方音乐的基础是 12 个半音，在吉他上，**相邻两品 = 1 个半音**。

```
C  C#  D  D#  E  F  F#  G  G#  A  A#  B  C
   Db     Eb        Gb     Ab     Bb
```

> `#` = 升半音，`b` = 降半音，同一个音可以有两个名字（等音异名）

---

## 三、大调音阶

### 公式
大调音阶 = **全 全 半 全 全 全 半**（W W H W W W H）

以 C 大调为例：
```
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
```

所以 C 大调没有任何升降号，是最容易入手的调。

### 在吉他上找 C 大调音阶（5弦根音 Am 把位）

```
e|---0---1---3---|
B|---0---1---3---|
G|---0---2------|
D|---0---2---3--|
A|---0---2---3--|
E|---1---3------|
```

---

## 四、和弦的构成

一个和弦最少需要 **3 个音**（三和弦）：

### 大三和弦（Major Chord）
公式：**根音 + 大三度（4 个半音）+ 小三度（3 个半音）**

以 C 大和弦为例：
- 根音：C
- 大三度：E（C 上数 4 个半音）
- 纯五度：G（E 上再数 3 个半音）

### 小三和弦（Minor Chord）
公式：**根音 + 小三度（3 个半音）+ 大三度（4 个半音）**

以 Am 为例：A + C + E

> **口诀**：大和弦明亮，小和弦忧郁。

---

## 五、调内和弦（最实用！）

每个调都有 7 个"天然"和弦，用罗马数字标记：

| 级数 | 音阶音 | C调 | 性质 |
|---|---|---|---|
| **Ⅰ** | C | C | 大和弦（主和弦） |
| **Ⅱ** | D | Dm | 小和弦 |
| **Ⅲ** | E | Em | 小和弦 |
| **Ⅳ** | F | F | 大和弦（下属和弦） |
| **Ⅴ** | G | G | 大和弦（属和弦） |
| **Ⅵ** | A | Am | 小和弦（关系小调） |
| **Ⅶ** | B | Bdim | 减和弦 |

### 最经典的进行

- **Ⅰ - Ⅳ - Ⅴ - Ⅰ**（C - F - G - C）：无数流行歌的基础
- **Ⅰ - Ⅴ - Ⅵ - Ⅳ**（C - G - Am - F）：卡农进行，流行程度最高
- **Ⅵ - Ⅳ - Ⅰ - Ⅴ**（Am - F - C - G）：同上，从 Ⅵ 级开始

## 六、变调夹（Capo）的用法

变调夹夹在某一品，相当于把整个琴"升调"。

| 变调夹位置 | 实际升高 | 常见用法 |
|---|---|---|
| 1 品 | 升半音 | 原 C 调 → C# 调 |
| 2 品 | 升 1 音 | 原 C 调 → D 调 |
| 3 品 | 升 1.5 音 | 原 C 调 → Eb 调 |
| 5 品 | 升 2.5 音 | 原 C 调 → F 调 |

**实用技巧**：遇到难按的 Bb、Eb 等调，先在 Capo 上找到用 C 调和弦能演奏的位置，再搞懂为什么。

---

## 七、五声音阶（速成即兴）

五声音阶只有 5 个音，用它即兴不容易出错，是摇滚/蓝调/民谣即兴的基础。

**A 小调五声（最常用把位）：**

```
e|---5---8------|
B|---5---8------|
G|---5---7------|
D|---5---7------|
A|---5---7------|
E|---5---8------|
```

在以 A 或 Am 为根音的歌上，用这个把位随意弹，基本都好听。

---

## 八、简单的和声分析步骤

当你听到一首歌，想知道它的和弦是什么：

1. 找出调性（听开头或结尾的和弦，通常就是主音）
2. 写出这个调的 Ⅰ - Ⅶ 七个和弦
3. 对照歌曲，找出哪些和弦和列表吻合
4. 用罗马数字标注（比如 Ⅵ - Ⅳ - Ⅰ - Ⅴ）
5. 这就是"级数"，换任何调都能用
