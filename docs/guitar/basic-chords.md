# 基础和弦

掌握这几个和弦，你就能弹唱 80% 的流行歌曲。

## 必学和弦（C调）

### 1. C 和弦

```
e|--0--|
B|--1--|  ← 食指
G|--0--|
D|--2--|  ← 中指
A|--3--|  ← 无名指
E|--x--|  (不弹)
```

**按法**：
- 食指按 2 弦 1 品
- 中指按 4 弦 2 品
- 无名指按 5 弦 3 品

### 2. G 和弦

```
e|--3--|  ← 小指
B|--0--|
G|--0--|
D|--0--|
A|--2--|  ← 中指
E|--3--|  ← 无名指
```

### 3. Am 和弦

```
e|--0--|
B|--1--|  ← 食指
G|--2--|  ← 中指
D|--2--|  ← 无名指
A|--0--|
E|--x--|
```

### 4. F 和弦（简化版）

初学者先用 **Fmaj7** 代替大横按：

```
e|--0--|
B|--1--|  ← 食指
G|--2--|  ← 中指
D|--3--|  ← 无名指
A|--x--|
E|--x--|
```

### 5. Em 和弦

最简单的和弦：

```
e|--0--|
B|--0--|
G|--0--|
D|--2--|  ← 中指
A|--2--|  ← 无名指
E|--0--|
```

### 6. Dm 和弦

```
e|--1--|  ← 食指
B|--3--|  ← 无名指
G|--2--|  ← 中指
D|--0--|
A|--x--|
E|--x--|
```

## 万能和弦进行

### 1. C - G - Am - F
最经典的进行，无数流行歌用这个：
- 《平凡之路》
- 《成都》
- 《小幸运》

### 2. Am - F - C - G
小调感更强，适合抒情歌：
- 《后来》
- 《遇见》

### 3. G - D - Em - C
G调常用，更明亮：
- 《拥抱》
- 《温柔》

## 和弦转换练习

### 每日必练（每个 1 分钟）

1. **C → G**（无名指从 5 弦 3 品移到 6 弦 3 品）
2. **G → Am**（整体下移，食指加在 2 弦）
3. **Am → Fmaj7**（保留食指，其他上移）
4. **Fmaj7 → C**（松开中指无名指，加无名指到 5 弦）

### 节拍器练习

- BPM 60：每和弦 4 拍
- BPM 80：每和弦 2 拍
- BPM 100：每和弦 1 拍（目标速度）

## 按弦技巧

1. **指尖按弦**：用指尖而不是指肚
2. **靠近品丝**：在品格靠近音孔一侧按，不是中间
3. **拇指位置**：放在琴颈背面中间，与中指相对
4. **手腕放松**：不要翻手腕去够弦

## 常见问题

| 问题 | 原因 | 解决 |
|-----|------|------|
| 有杂音 | 没按实或碰到其他弦 | 调整手指位置，单独检查每根弦 |
| 手指疼 | 正常，起茧就好了 | 每天练 20-30 分钟，不要过度 |
| 转换慢 | 肌肉记忆不够 | 慢速练，形成记忆后再加速 |
| 大横按不响 | 食指力度不够 | 先练 Fmaj7，再挑战 F |
