# MEMORY.md — Personal Web 项目长期记忆

## 项目概览
- **项目名**: 周云飞个人主页（个人博客 + 工具库导航）
- **位置**: `D:\code\Personal Web\`
- **主要文件**: `个人博客.html`（旧版）、`index.html`（新版工具库）、`style.css`

## 设计风格与约定
- 参考 maorx.cn 风格：简洁现代，无多余装饰
- 主色调：`#f7f7f5` 背景，`#1a1a1a` 文字，`#4f6ef7` 强调色
- 圆角：12px，阴影轻量，hover 微上浮 -2px
- 字体：系统字体栈（-apple-system, PingFang SC 等）

## 功能规划
- **工具库导航主页** (`index.html`)：以分类卡片展示好站
  - 分类：效率工具、学习平台、视频平台、书籍资源、设计资源
  - 有分类筛选 Filter 功能
  - 顶部 Ctrl+K 快速搜索弹窗
  - 精选区（本周精选）
- **旧版博客** (`个人博客.html`)：保留，结构已有：首页、关于我、作品、大事记、页脚
