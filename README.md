# 个人网站项目

> 一个完全静态、纯手写HTML/CSS/JS的个人网站，集知识库、摄影作品、工具导航于一体，展示个人技术积累和生活创意。

## 🌟 网站特色

### 🎯 核心定位
- **个人知识中心**：8大专业分类，55篇高质量技术文章
- **作品展示平台**：摄影相册、DIY项目、技术实践成果
- **工具资源库**：精心整理的6大类实用工具和网站导航

### 🚀 技术亮点
- **纯前端架构**：零框架依赖，原生HTML/CSS/JS实现
- **完全静态化**：所有内容内嵌，无需服务器，支持离线使用
- **响应式设计**：完美适配电脑、平板、手机等多种设备
- **极速加载**：前端实时搜索，无需网络请求，秒级响应

### 💡 设计理念
- **内容为王**：简洁界面突出内容，减少视觉干扰
- **易于维护**：纯静态文件结构，无复杂构建流程
- **开放共享**：所有代码和内容开源，便于学习交流

## 🚀 快速开始

### 本地预览（无需服务器）
- **直接双击** `knowledge.html` 即可在浏览器中打开知识库
- **零配置**：所有内容已内嵌在JavaScript中，无需任何环境配置
- **跨平台**：支持Windows、macOS、Linux，任何现代浏览器

### 在线部署

本项目完全静态化，支持主流静态托管平台：

- **GitHub Pages**：最简方案，push即部署，自动HTTPS
- **Cloudflare Pages**：免费全球CDN，全球加速
- **Vercel**：一键部署，自动CI/CD
- **腾讯云静态网站托管**：国内用户访问加速

**核心优势**：
- **零服务器依赖**：纯HTML/CSS/JS文件，无需后端
- **即时更新**：push到GitHub后1-5分钟自动生效
- **永久免费**：GitHub Pages等平台提供免费托管

**部署步骤**：
1. 将整个项目推送到GitHub仓库
2. 在仓库设置中开启GitHub Pages功能
3. 选择main分支，根目录 `/`
4. 分享生成的链接（如 `https://username.github.io/repository/`）

## 🔧 开发指南

### 新增文章
1. 在 `docs/{分类}/` 目录下创建 `.md` 文件
2. 文章会自动集成到知识库，无需额外配置

### 更新内容
1. 修改对应的 `.md` 文件
2. 刷新页面即可看到更新

### 部署更新
```bash
git add .
git commit -m "更新说明"
git push origin main
```

## 🧩 技术架构

### 核心技术
- **HTML5**：语义化标签，现代化语法，无障碍支持
- **CSS3**：CSS变量，Flexbox/Grid布局，响应式媒体查询
- **原生JavaScript**：ES6+语法，模块化设计，异步编程
- **Marked.js**：Markdown渲染引擎（CDN加载）

### 架构特点
- **纯前端**：零后端依赖，所有逻辑在浏览器中完成
- **零构建**：无需Webpack/Vite等构建工具，直接编辑部署
- **零框架**：无React/Vue/Angular等框架，保持代码轻量
- **内嵌内容**：所有文章内容嵌入JavaScript，实现离线使用

### 性能优化
- **前端搜索**：实时索引，无需网络请求
- **本地缓存**：利用浏览器缓存机制，提升二次访问速度
- **懒加载**：图片和视频按需加载，减少初始加载时间

## 🎯 设计原则

1. **简洁至上**：去除冗余视觉元素，专注于内容本身
2. **快速响应**：前端实时搜索，无需网络请求
3. **易于维护**：纯静态文件，无需复杂构建流程
4. **跨平台**：支持各种操作系统和浏览器
